# CreateGame — 短編RPGプロジェクト（Godot 4）

簡単な2D短編RPG（PC / Steam向け）のプロトタイプ雛形です。

## 概要
- エンジン: Godot 4
- 言語: GDScript
- 目標: 1〜3時間の短編RPG（MVP）

## 目次
- `scenes/` — サンプルシーン
- `scripts/` — GDScript コード
- `assets/` — アセット（仮）
- `docs/GDD.md` — ゲームデザインドキュメント（簡易版）
- `docs/story.md` — ストーリー（少年ジャンプ風）

## 実行方法
1. Godot 4 をインストール
2. このフォルダを Godot で開く（`project.godot`）
3. `Main.tscn` を実行して簡易プロトタイプを確認

### VSCode
- 推奨拡張: Godot Tools（`godotengine.godot-tools`）
- `.vscode/extensions.json` は推奨設定を含みます

このリポジトリの初期コミットはプロトタイプの素地（シーン・プレイヤー移動・GDD・ストーリー）を含みます。
