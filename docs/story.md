# ストーリー（少年ジャンプ風・短編）

主人公: ユウ（15歳）
ライバル→味方: カズマ（かつての敵）

## あらすじ
小さな港町で育ったユウは、ある日町を襲う謎の魔物に出会う。魔物討伐を志すが力不足で苦戦する中、強靭で冷酷な若き戦士カズマが現れる。最初、カズマはユウの邪魔をする“敵”として立ちはだかるが、共通の敵（本当の黒幕）が現れ、協力することでお互いの弱さと強さを知る。

二人は互いに認め合い、それぞれの夢・信念を語り合う。ユウは努力と仲間を信じる心で成長し、カズマはかつての孤独から他者を信じることを学ぶ。クライマックスで、彼らは力を合わせて黒幕を打ち倒し、勝利と友情を勝ち取る。

## テーマ
- 友情（仲間を信じる）
- 努力（成長の過程）
- 勝利（信念と連携の結果）

## シーン案（短編）
1. プロローグ: 港町の平穏 → 魔物襲来
2. 出会い: ユウとカズマの衝突
3. 共闘: 真の敵の出現、連携で敗北を喫すが撤退
4. 成長: 小クエストで力をつける、仲間が増える
5. クライマックス: 黒幕との決戦、カズマが味方になる
6. エピローグ: 町の平和と新たな旅立ち
