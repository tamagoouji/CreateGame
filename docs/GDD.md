# GDD（短縮版） — 小さな旅路（仮）

## コンセプト
短編（1〜3時間）2Dターン制RPG。探索・会話・バトルで物語を進める。主題は「友情・努力・勝利」。

## プラットフォーム
- Windows（Steam 向け）

## コア要素
- マップ: 3〜5程度（町 + 野外）
- 移動: キーボード／ゲームパッド対応
- 会話: NPC と対話→簡易クエスト
- 戦闘: ターン制（メニュー式）
- セーブ/ロード

## MVP 受け入れ基準
- マップ上で移動可能
- NPC と会話／クエスト受注→完了
- 1種の戦闘がプレイ可能（勝敗・報酬）
- Windows ビルドで起動可能

## 技術
- Godot 4 / GDScript

## 開発スプリント（最初の2週間）
- 準備: リポジトリ、プロジェクト雛形
- 移動基盤実装
- 会話／クエスト基礎
- 戦闘（ターン制コア）
- セーブ/ロードとビルド
